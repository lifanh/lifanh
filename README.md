# Hi, I’m Lifan

I’m a software engineer building small, practical tools around AI-agent workflows, observability, reusable agent skills, and quiet product interfaces.

My current work is centered on a focused toolkit:

* **[`pi-langfuse-extension`](https://github.com/lifanh/pi-langfuse-extension)** — observability for Pi Coding Agent runs.
* **[`skills`](https://github.com/lifanh/skills)** — reusable skills that help coding agents work better inside repositories.
* **[`quiet-paper`](https://github.com/lifanh/quiet-paper)** — a prose-first React/Tailwind design system for technical writing and lightweight app demos.
* **[`astro-quiet-paper`](https://github.com/lifanh/astro-quiet-paper)** — an Astro template for publishing technical writing, project notes, and demos with the Quiet Paper style.

I’m interested in making AI-agent systems easier to inspect, easier to guide, and easier to present as useful products.

## Featured Projects

### [Pi Langfuse Extension](https://github.com/lifanh/pi-langfuse-extension)

Observability for Pi Coding Agent.

This extension sends agent runs, model calls, tool executions, token usage, cost, and errors to Langfuse. It is designed with privacy in mind: metadata-only tracing by default, with opt-in capture for prompts, responses, and tool payloads.

### [Skills](https://github.com/lifanh/skills)

A personal collection of reusable agent skills.

The goal is to make coding agents more effective by giving them high-signal, task-specific operating knowledge. Current skills include repository guidance for `AGENTS.md` files and setup guidance for tools like browser-use.

### [Quiet Paper](https://github.com/lifanh/quiet-paper)

A quiet design system for personal tools, writing, and demos.

It provides React primitives, Markdown prose styling, Tailwind v4 tokens, and a restrained visual language for technical content: warm paper, ink-like type, hairline borders, and minimal chrome.

### [Astro Quiet Paper](https://github.com/lifanh/astro-quiet-paper)

A prose-first Astro template powered by `quiet-paper`.

It is designed for personal sites, technical writing, project documentation, and small interactive demos. It includes Markdown content, RSS, sitemap support, React islands, and Cloudflare deployment.
